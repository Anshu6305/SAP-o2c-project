# 🖼️ Screenshot Guide — SAP O2C Project

**Author:** Anshuman Routray | Roll No: 23052783 | KIIT University

Place your SAP screenshots in this folder following the naming convention below.

---

## Naming Convention

`[STEP_NUMBER]_[TCODE]_[description].png`

---

## Required Screenshots

| File Name | T-Code | What to Capture |
|---|---|---|
| `01_VA11_inquiry_creation.png` | VA11 | Inquiry creation screen with sold-to party and material filled |
| `02_VA21_quotation_creation.png` | VA21 | Quotation screen showing price, validity, and customer details |
| `03_VA01_sales_order.png` | VA01 | Sales Order overview with line items, net value, and delivery date |
| `04_VA01_atp_check.png` | VA01 | Availability Check (ATP) confirmation screen |
| `05_VL01N_delivery.png` | VL01N | Outbound Delivery creation screen with picking details |
| `06_VL02N_pgi.png` | VL02N | Post Goods Issue confirmation (after clicking Post Goods Issue button) |
| `07_VF01_billing.png` | VF01 | Billing document (Invoice F2) with net value and tax breakdown |
| `08_F28_payment_receipt.png` | F-28 | Incoming payment posting screen with AR clearing |
| `09_document_flow.png` | VA03 / VF03 | Document Flow screen showing the full chain from Inquiry to Payment |
